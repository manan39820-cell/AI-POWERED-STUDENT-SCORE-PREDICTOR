How It Works

Linear Regression Model

This predictor uses multiple linear regression to estimate exam scores based on study habits. The model is trained on historical student data and uses the Normal Equation from linear algebra.

β = (XᵀX)⁻¹Xᵀy

Where:

• β = coefficient vector (weights for each feature)

• X = feature matrix (study hours, attendance, etc.)

• y = target vector (actual exam scores)

• Xᵀ = transpose of X

• (XᵀX)⁻¹ = inverse of XᵀX
